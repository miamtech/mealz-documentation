## Item Selector

If you want to fully customize this component we recommend copying the code into your Mealz template configuration.
You can remove the code you do not override.

```kotlin
itemSelector{
    header {
         view = MyCustomItemSelectorHeaderView()
    }
    search {
         view = MyCustomItemSelectorSearchView()
    }
    selectedItem{
         view = MyCustomItemSelectorItemSelectedView()
    }
    success {
        view = MyCustomItemSelectorSuccessView()
    }
    loading {
         view = MyCustomItemSelectorLoadingView()
    }
    empty {
         view = MyCustomItemSelectorEmptyView()
    }
}
```
