Here is an example on how to inform the SDK whenever the user login state changes. We recommend using Observables
or EventListeners to that end.

```kotlin 
import ai.mealz.core.Mealz

class MealzAuth() {
  init {
    // CODE

    OBSERVABLE_ON_USER_OBJECT.collect { user ->
       Mealz.shared.user.updateUserId(userId = user.id)
    }
  }

  // CODE
}
```
